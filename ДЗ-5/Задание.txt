1.1. В карточку номенклатуры необходимо вывести остатки по каждому складу и организации аналогично ценам (использовать динамический список).
2.1. Создать запрос, который выводит самую дорогую номенклатуру на определенную дату (дата передается параметром).
2.2. Создать запрос, который отображает остатки номенклатуры с итогами по складу (дата остатков передается параметром). Остатки номенклатуры выбирается c отбором по организации (организация передается параметром).
2.3. Создать запрос, который выводит общую стоимость номенклатуры, хранящейся на каждом складе. То есть должно быть два столбца: "Склад" и "Стоимость". Дата расчета цен и остатков вводится одним параметром.
2.4. Модифицировать предыдущий запрос так, чтобы стоимость товаров по каждому складу рассчитывалась только для указанной в параметре организации.
